# Rodas Trial Consulting - Self Hosted Site

## Quick start (local)
1) Install Node.js (LTS)
2) In this folder:
   npm install
3) Create .env from .env.example and set passwords
4) Run:
   npm start
5) Open:
   Site:  http://localhost:3000
   Admin: http://localhost:3000/admin/login.html

## Admin
- Edit site title
- View contact form messages
